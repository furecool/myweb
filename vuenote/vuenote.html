• Vue.js筆記 
    為什麼要學框架(vue,react,angular)?
函式庫(jQuery) – 套別人寫好的動畫效果
jQuery – 用DOM元素來觸發功能、設計動畫效果
vue – 用資料來驅動功能
DOM – html階層樹中的元素(標籤<body><ul><li>)
This就是function執行時當下外層的物件

el:"#app",(eliment 哪個範圍要有vue功能)
data 資料綁定用
  • 1. 引入Vue庫 
    <head>
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
</head>
  • 2. 創建一個標籤 
    <body>
            <div id="app">
                    {{message}}
            </div>
          </body>
  • 3. 創建一個Vue對象 
    <body>…</body>
          <script type="text/javascript">

                   var app = new Vue({
                     el:"#app",    //vue對象的掛載點 
                     data: {       //自定義數據
                       message: "hello Vue!!",
                     },
                   });

                </script>
          
    連結
  • 4. 數據渲染 v-text、v-html、{{}} 
    data:{

          title:"<h2>指令學習</h2>",
          isShow: false,
          arr:['1.HTML','2.CSS','3.Javascript'],
          }

          <p v-text="title"></p>
          <p v-html="title"></p>
  • 5. 控制模塊顯示隱藏 v-if、v-show

    如果模塊不顯示，不渲染

    <p class="show" v-if="isShow"></p>

    模塊不顯示，還是渲染，只是藉由display顯示為none來隱藏

    <p class="show" v-show="isShow"></p>
  • 6. 渲染循環列表 v-for 
    <ul>
     <li v-for="item in arr">
       {{item}}
     </li>
</ul>

    連結

  • 7. 事件綁定 v-on 
    data:{

                    imgSite: 'img/safari.png',  //圖片位置
                    message: 'vue入門之模板指令學習',
                    isActive: true

          },
          methods:{

                    speak: function(){

                    alert("hello!!");

                    }
          }

            <p class="show" v-on:click="speak"></p>
            <p class="show" @click="speak"></p>
  • 8. 屬性綁定 v-bind : 
    <img  src="img/ppt.png">
<img  v-bind:src="imgSite">
<img  :src="imgSite">
  • 9. 樣式綁定 v-bind :

    isActive為true，給該div加個class叫active

    <div class="view" v-bind:class="{active:isActive}">
   {{message}}
</div>
